# Download And Run YouTube Summary App

This guide is for friends testing the app on a Mac.

## What This App Does

YouTube Summary App summarizes YouTube videos using a local AI model through LM Studio. The app itself is free and local, but it needs LM Studio running on your Mac to generate summaries.

## What You Need

- A Mac with Apple Silicon or a modern Intel Mac.
- LM Studio installed.
- One chat/instruct model downloaded in LM Studio.
- The YouTube Summary App DMG from the GitHub Release.

## Download The App

1. Open the GitHub Release link.
2. Under **Assets**, download:

```text
YouTube-Summary-App.dmg
```

3. Open the downloaded DMG.
4. Drag **YouTube Summary App** into **Applications** if shown, or copy it to your Applications folder.

## First Launch On macOS

This app is unsigned and not notarized because it is a free friend-test build. Apple may show a warning the first time you open it.

Try this first:

1. Open **Applications**.
2. Find **YouTube Summary App**.
3. Right-click or Control-click the app.
4. Click **Open**.
5. Click **Open** again if macOS asks.

If macOS still blocks it, use this command in Terminal:

```bash
xattr -dr com.apple.quarantine "/Applications/YouTube Summary App.app"
```

Then open the app again.

## Set Up LM Studio

1. Download and install LM Studio:

```text
https://lmstudio.ai/
```

2. Open LM Studio.
3. Download one chat/instruct model.
4. Load exactly one model.
5. Open LM Studio's **Developer** or **Local Server** area.
6. Start the local server on port:

```text
1234
```

7. Open YouTube Summary App.
8. Click **Test Connection**.
9. Paste a YouTube URL and click **Summarize Queue**.

## Recommended Models

For smaller Macs:

```text
Gemma 3 4B
Qwen 2.5 7B Instruct
Llama 3.2 3B Instruct
```

For Macs with more memory:

```text
Gemma 3 12B
Qwen 2.5 14B Instruct
GPT OSS 20B if your machine can handle it
```

Use quantized models if LM Studio offers them.

## Common Problems

### macOS says the app cannot be opened

Use right-click > **Open** first. If that does not work, run:

```bash
xattr -dr com.apple.quarantine "/Applications/YouTube Summary App.app"
```

### The app says LM Studio is not connected

Open LM Studio, load one model, and start the local server on port `1234`.

### The app says multiple models are loaded

Unload all but one model in LM Studio.

### The app is slow

Use a smaller model. Local AI speed depends on your Mac and the model size.

## Windows

This release is for macOS only. Windows users cannot run the `.dmg` or `.app`.

A Windows version would need a separate Windows build and testing pass.

## Privacy

The app is designed for local use:

- Summaries are generated by LM Studio on your machine.
- History is stored on your Mac.
- Chat history is stored on your Mac.
- The app does not include a cloud AI service.

YouTube transcript fetching still requires internet access.
