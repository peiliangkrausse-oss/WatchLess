DOWNLOAD AND RUN YOUTUBE SUMMARY APP

This guide is for friends testing WatchLess on a Mac.


1. WHAT THIS APP DOES

WatchLess summarizes YouTube videos using a local AI model through LM Studio.

The app is free and local. It does not include a cloud AI service and it does not include an AI model. To generate summaries, LM Studio must be installed and running on your Mac.


2. WHAT YOU NEED

- A Mac with Apple Silicon or a modern Intel Mac.
- LM Studio installed.
- One chat or instruct model downloaded in LM Studio.
- The WatchLess.dmg file from the GitHub Release.


3. DOWNLOAD THE APP

1. Open the GitHub Release link.
2. Scroll to Assets.
3. Download this file:

   WatchLess.dmg

4. Open the downloaded DMG.
5. Copy WatchLess to your Applications folder.

Do not download "Source code" unless you are a developer. Normal users need the DMG file.


4. FIRST LAUNCH ON MACOS

This is a free unsigned test build. Apple may warn that it cannot verify the app.

Try this first:

1. Open Applications.
2. Find WatchLess.
3. Right-click or Control-click the app.
4. Choose Open.
5. Confirm Open again if macOS asks.

If macOS still blocks it, open Terminal and run:

   xattr -dr com.apple.quarantine "/Applications/WatchLess.app"

Then open the app again.


5. SET UP LM STUDIO

1. Download LM Studio:

   https://lmstudio.ai/

2. Open LM Studio.
3. Download one chat or instruct model.
4. Load exactly one model.
5. Open LM Studio's Developer or Local Server area.
6. Start the local server on port 1234.
7. Open WatchLess.
8. Click Test Connection.
9. Paste a YouTube URL.
10. Click Summarize Queue.


6. RECOMMENDED MODELS

For smaller Macs, start with one of these:

- Gemma 3 4B
- Qwen 2.5 7B Instruct
- Llama 3.2 3B Instruct

For Macs with more memory:

- Gemma 3 12B
- Qwen 2.5 14B Instruct
- GPT OSS 20B, if your machine can handle it

Use quantized models if LM Studio offers them. Smaller models are usually better for first-time setup.


7. COMMON PROBLEMS

Problem: macOS says the app cannot be opened.

Use right-click > Open first. If that does not work, run:

   xattr -dr com.apple.quarantine "/Applications/WatchLess.app"


Problem: The app says LM Studio is not connected.

Open LM Studio, load one model, and start the local server on port 1234.


Problem: The app says multiple models are loaded.

Unload all but one model in LM Studio.


Problem: The app is slow.

Use a smaller model. Local AI speed depends on your Mac and model size.


8. WINDOWS

This release is for macOS only. Windows users cannot run the DMG or the Mac app.

A Windows version needs a separate Windows build and testing pass.


9. PRIVACY

The app is designed for local use:

- Summaries are generated by LM Studio on your machine.
- History is stored on your Mac.
- Chat history is stored on your Mac.
- The app does not include a cloud AI service.

YouTube transcript fetching still requires internet access.
