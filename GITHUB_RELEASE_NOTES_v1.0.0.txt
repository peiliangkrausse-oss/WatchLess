WATCHLESS V1.0.0

Badass release title:

LOCAL BRAIN, NO CLOUD: WatchLess v1.0.0


Short description:

WatchLess is a local Mac app that summarizes YouTube videos with LM Studio. Paste one or more YouTube links, queue summaries, save Markdown history, and chat with your local model about the result.

Important first launch note:

This release is intended to be signed and notarized so macOS can verify it.

To open it:

1. Move the app to Applications.
2. Double-click WatchLess.

If macOS says Apple could not verify the app, please report the exact message.


LM Studio is required:

This app does not include an AI model. It uses LM Studio as the free local AI runtime.

Before summarizing:

1. Install LM Studio from https://lmstudio.ai/.
2. Download one chat or instruct model.
3. Load exactly one model.
4. Start LM Studio's local server on port 1234.
5. Open this app and click Test Connection.


What's included:

- Paste one or more YouTube URLs.
- Queue multiple summaries.
- Summarize locally through LM Studio.
- Save Markdown history on your Mac.
- Chat with your local model about a summary.
- Edit and save prompt presets.
- Attach readable files to chat context.
- Use a built-in LM Studio setup guide.


Known limitations:

- macOS only.
- Requires LM Studio setup.
- Windows build is not included.
- Image upload is not supported yet.
- Summary speed depends on your Mac and model size.


Best model advice:

Start small:

- Gemma 3 4B
- Qwen 2.5 7B Instruct
- Llama 3.2 3B Instruct

If your Mac has more memory, try larger models later.
