GITHUB RELEASE CHECKLIST

Use this checklist to publish a free unsigned Mac build for friends.


1. BUILT ARTIFACT

Current local artifact:

   dist/WatchLess.dmg

Current checksum:

   d79490ace95d7265aa2d6dbd965459d20282653398f086edf7552df1a2064a4a


2. CREATE THE GITHUB RELEASE

1. Open the repository on GitHub:

   https://github.com/peiliangkrausse-oss/WatchLess

2. Click Releases.
3. Click Draft a new release.
4. Choose tag:

   v1.0.0

5. Release title:

   LOCAL BRAIN, NO CLOUD: WatchLess v1.0.0

6. Paste the contents of:

   GITHUB_RELEASE_NOTES_v1.0.0.txt

7. Under "Attach binaries by dropping them here or selecting them", upload:

   dist/WatchLess.dmg

8. Click Publish release.


3. FRIEND LINK

After publishing, send friends the release page link, not the repository source ZIP.

The link will look like:

   https://github.com/peiliangkrausse-oss/WatchLess/releases/tag/v1.0.0


4. SHORT MESSAGE TO SEND FRIENDS

Download WatchLess.dmg from Assets, not Source code.
Because this is a free unsigned Mac build, open it with right-click > Open the first time.
You also need LM Studio installed, one model loaded, and the local server running on port 1234.
Instructions are in DOWNLOAD_INSTRUCTIONS.txt.


5. IF MACOS BLOCKS THE APP

Tell testers to run:

   xattr -dr com.apple.quarantine "/Applications/WatchLess.app"

This is the free workaround when the app is not notarized through Apple's paid developer program.
